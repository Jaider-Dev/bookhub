## Bookhub - Documentación del Proyecto Final Desarrollo III

Este directorio contiene toda la documentación del proyecto en formato **Markdown (`.md`)**, cumpliendo lo solicitado en la rúbrica del documento `Proyecto final Desarrollo III.pdf`.

- `resumen-proyecto.md`: descripción del problema, objetivos, justificación y conclusiones.
- `arquitectura.md`: arquitectura de microservicios, patrones utilizados (API Gateway, etc.) y diagramas.
- `base-datos.md`: diseño de entidades, relaciones JPA y estructura mínima de 4 tablas.
- `api-rest.md`: descripción de los microservicios, endpoints principales y contratos DTO.
- `docker-y-despliegue.md`: explicación de los `Dockerfile`, `docker-compose.yml` y cómo ejecutar el sistema.

Para ejecutar todo el sistema con Docker:

```bash
docker compose up -d
```
